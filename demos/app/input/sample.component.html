Forms

Includes the following components and directives:

  • igx-radio allows single selection from a set.
  • igx-checkbox allows multiple selection from a set.
  • igx-switch turns on and off an option.
  • igx-input allows the user to enter a value into a field.
  • igxLabel gives a single line of text information. It is usually used in combination with igxbutton, igx-radio, igx-switch or igx-input.
